# Pack: Marketing

## Use when
- FB/Google/TikTok ads analysis
- Campaign performance checks
- Daily/weekly marketing reports

## Rules
- Focus on actionable insights (cost, leads, CPL/CPA, anomalies, next actions).
- Keep outputs concise; show numbers clearly.
- Flag abnormal trends early with suggested fix.
- If budget shift >20% or risky decision: request confirmation.

## Typical Skills
- `report-ads`
- `analyze-mess-ads`
- `search-kit` (research/benchmark)
- `progress-tracker` (if long workflow)
