# Pack: Automation & Engineering

## Use when
- Scripts, workflow automation, tooling setup
- Multi-step technical tasks
- Integrations and data pipelines

## Rules
- For tasks >3 steps, create/update a plan file.
- Verify by running checks/tests where possible.
- Prefer safe, reversible actions.

## Typical Skills
- `plans-kanban-lite`
- `test-orchestrator-lite`
- `workflow-test-checklist-lite`
- `error-translator`
- `daytona-sandbox` (untrusted code/heavy processing)
